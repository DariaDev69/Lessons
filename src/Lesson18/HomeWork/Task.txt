1. Создайте класс, в котором создайте метод sortirovka.
Инпут параметром данного метода будет одномерный массив типа int.
Метод должен возвращать уже отсортированный по возрастанию массив.
Продемонстрируйте метод.
2. Создайте класс, в котором создайте метод showArray.
Инпут параметром данного метода будет двумерный массив типа String.
Метод должен выводить на экран данный массив в следующем виде:
{{элемент00, элемент01},{элемент10},{элемент20, элемент21}}
Содержимое, естественно, будет зависеть от вашего массива.
Продемонстрируйте данный метод.
