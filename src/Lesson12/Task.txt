If-Else. Equels. Ternary.

Flow Control - контроль хода программы (возможны несколько вариантов хода программы - selection (выбор) и looping (зацикливание)).

Примеры selection это конструкции If-else и Switch.
Синтаксис конструкции if-else:

if (boolean условие) {тело, которое будет принято в работу только при условии TRUE}
else {тело, которое будет принято в работу при условии FALSE}

Метод Equels используется для сравнения обьектов по параметрам и этот метод есть у всех классов (он является дефолтным).
Пример использвание:
String s1 = new String ("abc");
String s2 = new String ("abc");
Но s1==s2 - всегда будет FALSE, так как это ссылки на 2 разных объекта.

НО!
s1.equels(s2) - в данном случае будет TRUE, так как содержимое одинаково.

Это пример метода Equels для класса String, в каждом созданном пользователем классе, если есть запрос на особую работу метода Equels,
требуется этот метод и его логику прописывать самостоятельно., т.е. требуется самостоятельно объяснить машине, что считать идентичными обьектами и по
 каким парамертам проводить сравнение.

Ternary оператор - используется крайне редко, но очень полезен.
СИнтаксис:
(boolean выражение) ? (if true) : (if false)
Пример:
int a = 10;
int b = 20;
int maximum = (a>b)?a:b
