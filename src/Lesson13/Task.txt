Switch statement.

Показывает обычно больше 2-х путей следование.

Синтаксис:
switch (выражение) {
case value1: our code; break;
case value2: our code; break;
...
default: our code; break; // Срабатывает тогда, когда не работает ни один из кейсов. После default break писать не обязательно.
//Также default можно написать где угодно, но если он не в конце, тогда надо написать break.

Кейс с одним и тем же значением написать нельзя!
Выражение - это не boolean выражение, это мб byte, short, int, char, String!
После ключевого слова case в value могу использовать только константы, которые определены во время компиляции: 1, -3.14, "privet", 5*10, т.е. обычные значения.
Нельзя использовать, например, a=5, a*10, a/b, даже если эти переменные определены изначально, т.к. компилятор не знает будут ли
переменные a и b изменены.
Но это можно исправить, сделав a и b константами и задать значение сразу при их объявлении.

}
