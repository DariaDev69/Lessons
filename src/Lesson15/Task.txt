Циклы while и do while.

While используется когда количество повторений цикла заранее не известно.
Это цикл с условием, т.е. сначала проверяется условие, потом только при TRUE совершается итерация.
Проверка условия происходит перед каждой итерацией.

Синтаксис While:

while (boolean условие) {
    our code;
}

Do ... while почти аналогичен While, только это цикл с постусловием, т.е. происходит итерация и в конце совершается проверка условия,
и при условии TRUE происходит следующая итерация, и т.д., пока не будет FALSE.

Синтаксис Do while:

do {
    our code;
} while (boolean условие);
